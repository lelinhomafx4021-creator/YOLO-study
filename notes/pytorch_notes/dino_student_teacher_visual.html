核心想法

同一张图的不同裁剪 → 特征应该相似。
不同图的裁剪 → 特征应该不同。
不需要任何标签。

Loss = CrossEntropy(输出A, 输出B)
让 A 和 B 尽量一致

为什么需要两个网络

如果只有一个网络，它会学到"不管输入什么，都输出一样的特征"。
这样 Loss=0，但什么都没学到（坍缩）。

❌ 只有一个网络
"不管输入什么，都输出 [0.5, 0.5, 0.5]"
Loss=0，但什么都没学到

Teacher 是 Student 的 EMA（指数移动平均）。
Student 在努力学习，Teacher 是 Student 的"慢版本"，提供稳定的目标。

Teacher = 0.996 × Teacher + 0.004 × Student
每步更新一点点，比 Student 稳定
