# Transformer Encoder 完整结构

## 这一节在学习顺序里的位置

上一节你已经知道：

- 自注意力机制：Q·K 算权重，权重加权 V 得到输出
- 每个词的新表示 = 所有词按关注度混合

但完整的 Transformer Encoder 不只是自注意力。它还有一堆"辅助结构"。

这一节讲清楚：**一个 Transformer Encoder Block 到底由哪些部分组成，每个部分干什么。**

---

## 先说最核心的一句话

**一个 Encoder Block = 自注意力 → 残差连接 → LayerNorm → 全连接 → 残差连接 → LayerNorm。就这六步，重复 N 次。**

---

## 1. 整体结构（先看全景）

```
输入
  ↓
┌─────────────────────────────────┐
│  Multi-Head Self-Attention      │ ← 自注意力（多个头并行）
│  + 残差连接 (Residual)          │ ← 输出 = 注意力结果 + 原始输入
│  + Layer Normalization          │ ← 归一化
├─────────────────────────────────┤
│  Feed Forward Network           │ ← 全连接层（两层 MLP）
│  + 残差连接 (Residual)          │ ← 输出 = FFN结果 + 注意力结果
│  + Layer Normalization          │ ← 归一化
└─────────────────────────────────┘
  ↓
输出
```

这就是一个 Encoder Block。堆 6 个或 12 个这样的 Block，就是完整的 Transformer Encoder。

---

## 2. 多头注意力（Multi-Head Attention）

上一节讲的自注意力是"一个头"。多头注意力就是**同时跑多个自注意力，然后拼起来**。

```
单头注意力:
  一个 Q、一个 K、一个 V → 一组注意力权重 → 一个输出

多头注意力 (比如 8 个头):
  头1: Q1, K1, V1 → 注意力1 → 输出1
  头2: Q2, K2, V2 → 注意力2 → 输出2
  ...
  头8: Q8, K8, V8 → 注意力8 → 输出8
  
  拼接: [输出1, 输出2, ..., 输出8] → 线性变换 → 最终输出
```

**为什么需要多头？**

一个头只能学一种注意力模式。多个头可以同时学不同的模式：

```
头1 可能学到: 关注前一个词（语法关系）
头2 可能学到: 关注很远的词（语义关系）
头3 可能学到: 关注同类词（实体关系）
```

**多头 = 多角度理解同一句话。** [看多头注意力可视化](multihead_attention_visual.html)

---

## 3. 残差连接（Residual Connection）

残差连接就做一件事：**把输入直接加到输出上。**

```
输出 = SelfAttention(输入) + 输入
                    ↑              ↑
               注意力的结果      原始输入（直接加过来）
```

**为什么要加？**

深度网络训练时，梯度经过很多层会衰减（梯度消失）。残差连接提供了一条"捷径"，让梯度可以直接跳过这一层传回去。

```
没有残差: 梯度要经过注意力层 → 越传越弱 → 深层训练不动
有残差:   梯度走捷径直接回去 → 不衰减 → 深层也能训练
```

**残差连接让 Transformer 能堆很多层（6/12/24 层）而不训练不动。**

---

## 4. Layer Normalization

LayerNorm 做的事情：**把向量的值标准化到均值=0、方差=1。**

```
输入: [3.2, -1.5, 0.8, 2.1]
均值 = 1.15, 标准差 = 1.72

标准化: (每个值 - 均值) / 标准差
输出: [1.19, -1.54, -0.20, 0.55]
→ 均值=0, 方差=1
```

**为什么要标准化？**

没有 LayerNorm 时，每一层的输出值范围会越来越大或越来越小，训练不稳定。LayerNorm 把值拉回合理范围。

**和 BatchNorm 的区别：**

BatchNorm 对一个 batch 内同一维度做标准化（依赖 batch 大小）。
LayerNorm 对单个样本的所有维度做标准化（不依赖 batch 大小）。

Transformer 用 LayerNorm 因为序列长度不固定，BatchNorm 不好用。

---

## 5. Feed Forward Network（前馈网络）

自注意力之后，每个词的向量再过一个两层的全连接网络：

```
输出 = ReLU(输入 × W1 + b1) × W2 + b2
```

```
输入: 512维 → W1 → 2048维 → ReLU → W2 → 512维
                   ↑ 先放大        ↓ 再缩回来
```

**为什么注意力之后还要过 FFN？**

自注意力做的是"词与词之间的关系"——让每个词吸收其他词的信息。
FFN 做的是"每个词自己的变换"——对吸收完信息后的表示做进一步加工。

**类比：自注意力是"开会讨论"，FFN 是"会后自己消化"。**

---

## 6. 位置编码（Positional Encoding）

Transformer 的自注意力不关心顺序。

```
"我 去 北京" 和 "北京 去 我"
→ 自注意力算出来的结果完全一样！
```

因为注意力只看点积，点积不关心向量的排列顺序。

**位置编码就是给每个位置加一个"位置信号"，让 Transformer 知道词的顺序。**

```
词向量 + 位置编码 = 带位置信息的输入

"我"  在位置0 → 加 [sin(0), cos(0), sin(0), ...]
"去"  在位置1 → 加 [sin(1), cos(1), sin(1/100), ...]
"北京"在位置2 → 加 [sin(2), cos(2), sin(2/100), ...]
```

原始论文用的是正弦/余弦函数。ViT 用的是可学习的位置编码（训练时自动学）。

---

## 7. 完整的一层 Encoder Block

把上面所有组件串起来：

```
输入 X (带位置编码)
  ↓
Multi-Head Self-Attention(X) → attn_out
  ↓
attn_out + X  (残差连接)
  ↓
LayerNorm(attn_out + X) → norm1
  ↓
FFN(norm1) → ffn_out
  ↓
ffn_out + norm1  (残差连接)
  ↓
LayerNorm(ffn_out + norm1) → 输出
```

**这就是一个 Encoder Block。重复 6 次（或 12 次）就是完整的 Transformer Encoder。**

---

## 8. 和 CNN 的对比

CNN 一个卷积层做的事情：

```
3×3 卷积 → ReLU → 输出
```

Transformer 一个 Encoder Block 做的事情：

```
自注意力 → 残差 → LayerNorm → FFN → 残差 → LayerNorm
```

**Transformer 的单层比 CNN 复杂得多，但每一层能学到更全局的信息。**

---

## 9. 这一节最该记住的 5 句话

1. **一个 Encoder Block = 自注意力 + 残差 + LayerNorm + FFN + 残差 + LayerNorm**
2. **多头注意力 = 多个自注意力并行，每个头学不同的注意力模式**
3. **残差连接 = 输入直接加到输出，解决梯度消失，让网络能堆很深**
4. **LayerNorm = 把向量标准化到均值=0方差=1，稳定训练**
5. **FFN = 对每个词自己的表示做进一步加工（"会后消化"）**

---

## 10. 学完这一节后你应该能回答的问题

**Q1: 一个 Transformer Encoder Block 由哪些部分组成**

A: **自注意力 → 残差连接 → LayerNorm → FFN → 残差连接 → LayerNorm。**

就这六步，重复 N 次（6/12/24 层）就是完整的 Transformer Encoder。

---

**Q2: 多头注意力是什么，为什么需要多头**

A: **同时跑多个自注意力，每个头学不同的注意力模式。**

一个头只能学一种关系（比如语法关系）。多个头可以同时学多种关系（语法、位置、语义）。最后拼起来，信息更丰富。

---

**Q3: 残差连接解决什么问题**

A: **梯度消失。**

深度网络训练时，梯度经过很多层会衰减。残差连接提供了一条"捷径"，让梯度可以直接跳过这一层传回去。让 Transformer 能堆很多层而不训练不动。

---

**Q4: LayerNorm 和 BatchNorm 的区别是什么**

A: **BatchNorm 对一个 batch 内同一维度做标准化（依赖 batch 大小）。LayerNorm 对单个样本的所有维度做标准化（不依赖 batch 大小）。**

Transformer 用 LayerNorm 因为序列长度不固定，BatchNorm 不好用。

---

**Q5: FFN 在 Transformer 里起什么作用**

A: **对每个词自己的表示做进一步加工。**

自注意力做的是"词与词之间的关系"（开会讨论）。FFN 做的是"每个词自己的变换"（会后消化）。两者缺一不可。

---

**Q6: 为什么 Transformer 需要位置编码**

A: **因为自注意力不关心顺序。**

"我 去 北京"和"北京 去 我"，自注意力算出来的结果完全一样。位置编码给每个位置加一个"位置信号"，让 Transformer 知道词的顺序。

---

**Q7: 完整的数据流是什么**

A: 输入 X（带位置编码）→ 多头自注意力 → +X（残差）→ LayerNorm → FFN → +残差 → LayerNorm → 输出。重复 6-12 层。

---

## 复习速答

- `Encoder Block`：多头注意力 + 残差 + LayerNorm + FFN。
- `残差连接`：缓解梯度消失。
- `LayerNorm`：对单个样本做归一化。
- `FFN`：对每个位置做进一步非线性变换。
- `位置编码`：告诉模型顺序。
