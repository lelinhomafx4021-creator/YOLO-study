## 🎬 一、 视频流核心控制（VideoCapture）

把它当成一个“连环画翻页机”：

### 1. `cv2.VideoCapture(source)` —— 开启翻页机
*   **用途**：打开视频源，准备读取。
*   **参数 `source`**：
    *   填 `0`：打开电脑默认的前置 USB 摄像头。
    *   填 `"视频路径.mp4"`：打开本地的一个视频文件。
    *   填 `"rtsp://..."`：打开网络监控流。
*   **示例**：`cap = cv2.VideoCapture("test.mp4")`

### 2. `cap.read()` —— 撕下一页
*   **用途**：按顺序读取一帧画面。
*   **返回值**：它会返回**两个值**！`ret, frame = cap.read()`
    *   `ret`：一个布尔值（True/False）。如果成功撕下一页，就是 True；如果视频放完了（没图了），就是 False。通常用 `if not ret: break` 来判断视频是否结束。
    *   `frame`：这是真正的图像数据。它是一个 NumPy 的三维矩阵（高、宽、颜色通道 BGR）。

### 3. `cap.release()` —— 拔掉插头
*   **用途**：释放视频文件或关闭摄像头。
*   **注意**：程序结束前**必须调用**！如果不写，你的摄像头灯会一直亮着，或者视频文件被一直占用（删都删不掉）。

## 🎨 二、 在图片上涂鸦（Drawing API）

OpenCV 画图的函数都会**直接修改**你传进去的那张图片，不需要 `img = cv2.xxx()` 这样重新赋值。

### 1. `cv2.rectangle(img, pt1, pt2, color, thickness)` —— 画方框
*   **用途**：在图片上画一个矩形框（最常用于画检测框）。
*   **参数**：
    *   `img`：你要画的图片，比如 `frame`。
    *   `pt1`：左上角坐标，必须是元组，且必须是整数 `(x1, y1)`。
    *   `pt2`：右下角坐标，必须是元组 `(x2, y2)`。
    *   `color`：颜色！⚠️ 注意 OpenCV 里是 **BGR 顺序**（不是 RGB）。
        *   `(0, 0, 255)` = 红色报警框
        *   `(0, 255, 0)` = 绿色安全框
        *   `(255, 0, 0)` = 蓝色
    *   `thickness`：线条的粗细。比如填 `2`。如果填 `-1`，就会画一个实心的实心块。

### 2. `cv2.putText(img, text, org, font, fontScale, color, thickness)` —— 写文字
*   **用途**：在图片上写英文、数字（比如 `FPS: 30` 或 `helmet 95%`）。注：OpenCV 原生不支持写中文。
*   **参数**：
    *   `text`：要写的字符串。
    *   `org`：文字起始位置的**左下角**坐标 `(x, y)`。
    *   `font`：字体风格，无脑用 `cv2.FONT_HERSHEY_SIMPLEX` 即可。
    *   `fontScale`：字体大小缩放比例，比如 `0.8` 或 `1.0`。
    *   `color` / `thickness`：颜色（BGR）和粗细。

## 💾 三、 文件存取与 NumPy 抠图

### 1. `cv2.imread(filepath)` 和 `cv2.imwrite(filepath, img)`
*   **`imread`**：从硬盘读取一张图片进来。
*   **`imwrite`**：把内存里的 `img`（可以是被你画过框的图）保存到硬盘上。
    *   **示例**：`cv2.imwrite("result.jpg", frame)`

### 2. ✂️ NumPy 切片抠图（高危抓拍绝技）
OpenCV 的图像 `frame` 本质上是个大数组。要想从图里扣出一个人脸，不需要调函数，直接用 Python 的切片即可：
*   **语法**：`crop_img = frame[y_start : y_end, x_start : x_end]`
*   **⚠️ 避坑提醒**：
    1.  注意切片的顺序，**一定是先 y（行数/高度），再 x（列数/宽度）**。
    2.  切出来的 `crop_img` 可以直接传给 `cv2.imwrite()` 保存成大头照。

## 📺 四、 弹窗展示与魔法限速器

### 1. `cv2.imshow(winname, img)` —— 投射展示
*   **用途**：在电脑屏幕上弹出一个窗口，把你指定的图片展示出来。
*   **参数 `winname`**：窗口的标题名字，比如 `"My Video"`。

### 2. `cv2.waitKey(delay)` —— 起搏器与限速器
*   **用途**：必须和 `imshow` 绑定使用！它负责刷新窗口，并暂停程序等待用户敲击键盘。
*   **参数 `delay`**：等待多少毫秒。
    *   `waitKey(1)`：只等 1 毫秒，然后迅速去执行 `while` 循环的下一行。用于播放视频（看起来最顺滑）。
    *   `waitKey(0)`：代码会死死卡在这一行无限等待，直到你按了键盘任何一个键。用于查看单张图片。
*   **经典退坑组合拳**：
    ```python
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break # 如果你在键盘上按了 'q'，跳出循环，结束播放
    ```

### 3. `cv2.destroyAllWindows()` —— 砸毁放映机
*   **用途**：程序结束前，把所有用 `imshow` 弹出来的窗口强行关闭。不写的话，你的窗口死活关不掉。

## 🎞️ 五、 视频合成器（VideoWriter）

如何把你一帧一帧画好框的图片，拼装成一个能发给老板看的 `.mp4` 视频？

### 1. `cv2.VideoWriter_fourcc(*'mp4v')`
*   **用途**：设置视频编码格式。背下来就行，对于 mp4 视频，填 `*'mp4v'` 是最万能的。

### 2. `cv2.VideoWriter(filename, fourcc, fps, frameSize)`
*   **用途**：创建一个“空相册”，准备往里塞处理好的图片。
*   **参数**：
    *   `filename`：输出的名字，比如 `"result.mp4"`。
    *   `fps`：你希望这个视频播多快？通常填 `30.0`。
    *   `frameSize`：画面分辨率 `(width, height)`，必须和你要塞进去的 `frame` 的宽高一模一样，错一个像素都会保存失败，且不会报错（最终得到一个 0kb 的损坏文件）。
*   **塞入方法**：在循环里，调用 `out.write(frame)` 往里面加图片。
*   **结束动作**：循环结束后，别忘了调用 `out.release()` 把相册封上皮！
